# Organizador de Notas

Projeto 2 da minha jornada builder com IA.

## Objetivo

Criar uma mini CLI local em Python para salvar, listar e ler notas de estudo ou pesquisa.

## Objetivo didático

Aprender persistência de dados com arquivos.

## MVP 0.1

O programa deve:

- pedir um título de nota;
- pedir o conteúdo da nota;
- criar a pasta `notas/`, caso ela não exista;
- salvar a nota como arquivo `.md`.

## Como rodar

```bash
python main.py
